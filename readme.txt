JOBSKART
A dynamic website using HTML, CSS, NodeJS in SheHacks 5.0. The idea of the project was to
provide information about jobs by specifying job title and location. It fetches job data through API and also
provide links to apply for that particular job.

We were one of top3teams among freshers and top10 among 70+ teams participating all over India in SheHacks2022
